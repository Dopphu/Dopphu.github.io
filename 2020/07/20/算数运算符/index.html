算数运算符

运算符：

​ 1.算数运算符

​ + - * / %（取余）

​ 2.关系运算符

​ > < >= <= == != ===(恒等) !==

​ 3.逻辑运算符

​ || && !

​ 4.一元运算符

​ ++ –

​ 5.赋值运算符

​ <1>基本赋值运算符

​ <2>复合赋值运算符 += -= … *=

表达式：任何数据和运算符组成的式子

自动数据类型转换：不同数据类型之间是没有办法进行运算，将数据转换为同一类型，再进行运算。

​ 1.其中有一个操作数必须为字符串，运算符必须是+号，别的会会转换为字符串，再进行字符串拼接。

​ 2.如果字符串中含有字符串以外的别的字符，转成NaN，NaN和任何数据类型运算都是NaN

​ 3.除了字符串以外的数据，在进行算数运算的时候，先转成数字，在进行运算。

JS中除数可以为0，会得到

infinity（无穷大）-infinity（无穷小）
